# Salon

ERM for Nail Salons

### Dependencies

1. Golang 1.20+
2. pnpm
3. React

### To Run

1. Frontend

```bash

cd salon-frontend

pnpm install

pnpm run

```

2. Backend

```bash

cd salon-backend

./scripts/install-services.sh

make all

```
